# Support

Wuwa ini Tool is maintained by volunteers and has no guaranteed response time,
service-level agreement, or individual performance-tuning service.

## Before opening an Issue

1. Use the latest verified release or identify the exact commit you built.
2. Read [README.md](README.md), [DISCLAIMER.md](DISCLAIMER.md), and the in-app
   warning for the feature involved.
3. Restore a verified backup if an `Engine.ini` change caused a problem.
4. Reproduce with one change at a time and record both positive and negative
   results.
5. Remove account identifiers, personal paths, access tokens, and unrelated log
   content before sharing artifacts.

Use the repository's structured forms:

- **Bug report** for reproducible application defects.
- **Option evidence** for claims that an INI option works, is ignored, or has
  regressed.
- **Feature request** for a scoped product proposal.
- **Private security advisory** for vulnerabilities, as described in
  [SECURITY.md](SECURITY.md).

## What the project cannot promise

Maintainers cannot guarantee an FPS improvement, diagnose every game or driver
crash, prevent game updates from overriding settings, recover every damaged
configuration, or determine how a game operator will treat third-party tools.
The project cannot support modified clients, anti-cheat bypasses, pirated
copies, injection, memory editing, registry-wide optimizer packs, or unrelated
Windows debloating.

NVIDIA cache cleanup affects a driver-wide current-user cache, not only WuWa.
Locked files may remain, other applications may rebuild shaders, and the first
run can stutter. This is troubleshooting, not a routine FPS guarantee.

## Community safety

Do not ask contributors to upload full configuration directories, save data,
account files, private crash dumps, or updater signing material. Keep discussion
respectful and follow [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
